# Ponnect Mobile App

React Native mobile application for Ponnect - Australian Dog Parent Community.

## Features

- Community forums
- Expert Q&A
- Local events
- Pet health tracking
- Cross-platform (iOS & Android)

## Setup

```bash
npm install
npm start
```

## Running

- iOS: `npm run ios`
- Android: `npm run android`
- Web: `npm run web`
